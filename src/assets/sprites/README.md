# 캐릭터 스프라이트 에셋

플레이어(화랑) 캐릭터 스프라이트를 레벨 구간별로 나눠 보관한다.

| 폴더 | 대상 레벨 | 설명 |
|------|-----------|------|
| `child/` | Lv 1 ~ 10 | 꼬마 화랑 |
| `boy/` | Lv 11+ | 소년 화랑 |

`playerData.js`의 레벨에 따라 두 세트가 자동 전환된다 (전환 시 현재 애니메이션 상태 유지).

## 필요한 애니메이션 상태 (5종)

| 상태 | 키 | 트리거 |
|------|-----|--------|
| 정지 | `idle` | 적이 없어 멈춰 있을 때 |
| 이동 | `run` | 적을 향해 이동 중 |
| 점프 | `jump` | 점프 중 (공중) |
| 공격 | `attack` | 적 타격 |
| 피격 | `hurt` | 적에게 피해를 입을 때 |

## 파일 형식 (아래 중 택1 — 실제 파일 형태에 맞춰 로딩 코드 작성)

- **(A) 상태별 개별 시트**: `idle.png`, `run.png`, `attack.png`, `hurt.png`, `jump.png`
  각 파일은 가로로 프레임이 나열된 스트립 형태.
- **(B) 단일 통합 시트**: 모든 프레임이 하나의 격자에 들어간 형태.
- **(C) 아틀라스**: `sheet.png` + `sheet.json` (TexturePacker 등). 가장 정확함.

> 실제 PNG를 이 폴더에 넣으면 프레임 크기를 확인해 로딩·애니메이션 코드를 연결한다.
