# 🐉 요괴록 (妖怪錄) — 개발 문서

> 한국 요괴가 주인공인 방치형 육성 게임 (디지몬 키우기 × 메이플 키우기 스타일)
> 1인 개발 / 한국 설화·요괴 고증 기반 프로젝트
>
> *※ "요괴록"은 작업용 제목(옛 "군웅록"에서 전환). 최종 타이틀은 확정 전.*

---

## 📌 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 장르 | 방치형 요괴 육성 + 액션 플랫포머 (Idle Monster-Raising) |
| 컨셉 | **한국 요괴가 주인공, 인간이 적** — 인간을 포식해 정기를 모아 진화·승천 |
| 플랫폼 | 웹 먼저 출시 (itch.io) → 모바일 앱 (Capacitor 변환) |

| 레퍼런스 | 디지몬 키우기(분기 진화·육성·도감) + 메이플 키우기(방치형 자동 전투) |

---

## 💡 컨셉 & 차별화

- **주인공은 한국 요괴, 인간이 악당** — 승천을 위해 인간을 사냥하는 포식자 판타지
- **디지몬식 분기 진화** — 같은 요괴라도 **어떻게 키웠느냐**에 따라 다른 최종체로 갈림
- **한국 요괴·설화 고증** — 중국/일본 요괴물이 포화된 시장에서 한국 요괴(이무기·구미호·도깨비 등)는 미개척 IP. 진화 라인·형태·설정을 설화 기반으로 구성
- **방치형 + 육성 혼합** — 손 놓아도 자동 전투로 성장하되, 먹이·훈련으로 "키우는" 능동적 재미

---

## 🎮 핵심 게임 루프

```
[알 부화 / 요괴 선택]
    ↓ 이무기 · 구미호 등 요괴 1종
[고정 맵 자동 전투]
    ↓ 요괴가 가장 가까운 인간을 자동 사냥 (이동·점프·공격)
    ↓ 인간 병사는 맵에 계속 리스폰, 스테이지마다 강해짐
[포식 → 성장]
    ↓ 처치 시 경험치(레벨) · 정기(진화) · 골드(강화) 획득
[육성 (키우기)]
    ↓ 먹이·훈련으로 컨디션·훈련치 관리 → 진화 방향 결정
[분기 진화 / 승천]
    ↓ 정기 충만 시 진화, 육성 방향에 따라 최종체가 갈림
    ↓ 도달한 형태를 도감에 등록
[스테이지 진행]
    ↓ 인간 마을 → 관군 → 도성 → 천계의 문 (보스: 인간 장수)
[전생 (환생)]  ※ 예정
    ↓ 최종 진화 후 알로 회귀, 영구 배율 + 다른 갈래 재도전
```

> 맵 구조: **무한 스크롤 아님.** 고정 크기 맵 안에서 요괴가 자유 이동하며 싸운다 (메이플 사냥터 방식).

## 🐲 요괴 종족 & 분기 진화 트리

요괴는 **직업을 대체하는 정체성**이며, 각자 고유한 분기 진화 트리를 가진다.
데이터(`src/game/species.js`)로 정의되어 종족·갈래 추가가 쉽다.

### 이무기 (구현됨)
```
새끼 이무기 ──▶ 이무기 ──┬─▶ 흑룡  (공격 특화 시)
                         ├─▶ 청룡  (체력 특화 시)
                         └─▶ 황룡  (기본)
```

### 구미호 (데이터 존재, 선택 UI 예정)
```
여우 ──▶ 구미호 ──┬─▶ 흑호 (공격) / 백호 (체력) / 천호 (기본)
```

### 진화 규칙
- **정기**를 모아 진화 (형태마다 필요량 상승, 최종체 = 승천)
- **분기 조건**: 육성 방향(훈련/투자)에 따라 갈래가 잠금/해금
- 진화 시 해금된 갈래를 **선택**, 잠긴 갈래는 조건(`???`)으로 표시
- 진화할수록 능력치 배율 상승 (예: 이무기 ×1.9 → 흑룡 ×3.8)

### 향후 요괴 후보
도깨비 · 불가사리 · 그슨대 · 어둑시니 등 (각자 진화 라인 + 설화 설정)

## 🍖 육성(케어) 시스템

방치형 자동 전투 위에 얹는 **능동적 "키우기" 레이어.** 진화 방향을 결정한다.

- **먹이 주기** — 골드로 구매, 포만감 회복
- **훈련** — 포만감(에너지)을 소모해 공격형/체력형 훈련치 상승 → **진화 갈래를 유도**
- **컨디션(포만감)** — 시간이 지나면 감소, 방치하면 탈진 → 성장 효율 하락
- 즉 "무엇을 먹이고 어떻게 훈련했느냐"가 **어떤 요괴로 진화하는지**를 바꾼다 (디지몬 육성 핵심)

## 📅 리텐션 메타 (구현됨)

매일 돌아올 이유를 만드는 참여 유도 레이어. `retentionData` 순수 모듈 + 좌측 패널(탭).

- **일일 미션** — 매일 0시 갱신, 풀에서 3개 순환 선택(처치·정기·골드·보스·원정·진화). 달성 시 골드/재료/정기 보상 수령
- **업적(도전과제)** — 15종 영구 목표(레벨·챕터·전생·도감·누적 처치 등 티어별). 최고 기록으로 진행, 1회 수령
- **출석 보상** — 7일 주기 연속 출석(끊기면 리셋). 7일차 큰 보상. 하루 1회 수령
- 수령 가능 보상은 토글 버튼에 **빨간 배지 숫자**로 표시 → 패널을 열게 유도
- 서버/로컬 저장에 진행·수령 상태 영속화 (날짜 기준 갱신)

## 💰 방치형 메타 (구현됨)

- **골드** — 인간 처치 시 드랍 (스테이지·업그레이드 배율 반영)
- **업그레이드 상점** — 공격력 / 체력 / 골드 획득량 강화 (영구, 골드 소비)
- **오프라인 보상** — 자리를 비운 시간 × 획득 속도 × 효율(50%, 최대 8시간)

## 💾 저장 / 백엔드 (구현됨)

- **Spring Boot + SQLite** — 전체 저장(레벨·스테이지·골드·업그레이드·진화·도감) 영속화
- 프론트 `saveManager` 가 부팅 시 복원 + 변경/주기/종료 시 저장
- **백엔드 미기동 시 localStorage 폴백** → 프론트 단독 실행 가능
- 저장이 JSON 구조라 새 데이터(진화·도감·육성) 추가 시 백엔드 무변경

## 🛠️ 기술 스택

### Frontend
- **React** — UI (HUD, 상점, 도감, 진화 패널, 오프라인 모달)
- **Phaser.js (Arcade Physics)** — 요괴 이동·점프·충돌·전투

### Backend
- **Spring Boot (Java 21)** — 게임 데이터 저장 API
- **SQLite** → 추후 확장 시 이전 검토

### 배포
- **itch.io** — 웹 초기 출시
- **Capacitor.js** — 모바일 앱 변환 (iOS / Android)

### 아키텍처 원칙
- 게임 로직(Phaser)과 데이터 모듈(`src/data/`)을 분리 — 구독 패턴, 저장 교체 용이
- 콘텐츠(요괴·스테이지·업그레이드)는 데이터로 정의 — `species.js`, `stages.js`, 상수 파일

### ✅ 완료
- React + Phaser + Vite 세팅
- 고정 맵 자동 전투 (가장 가까운 인간 자동 사냥)

- 요괴 컨셉 전환 (요괴 주인공 / 인간 적)
- **분기 진화 트리 + 진화 조건 + 도감** (7종 × 5형태 = 35형태) + 수집 보상
- **육성(케어) 시스템** — 먹이 / 훈련 / 컨디션 → 진화 방향 결정
- **전생(환생) 프레스티지** — 최종 진화 후 알로 회귀, 영구 배율
- **요괴 종족 선택 / 알 부화 UI**
- **스킬 시스템** (형태별 고유기 — 단일/광역/브레스)
- **원정** (비활성 요괴 재료 수급) + 재료 제단·진화 촉진·스킬 강화
- **도트(픽셀아트)** — 코드 생성 스프라이트 (종족별 × 형태별(tier) 차등 + 인간 병사/장수)
- **디자인** — 밤하늘 야경, 타격/처치 이펙트, 먹색·금박 UI 테마
- **사운드** — Web Audio 절차적 합성 효과음(공격·포식·진화·스킬·보스 등) + 은은한 BGM + 음소거/볼륨 설정
- **리텐션 메타** — 일일 미션(매일 3개 순환) · 업적(15종 영구 도전과제) · 출석 보상(7일 주기 연속 출석)
- **출시 준비** — 타이틀 확정(요괴록/妖怪錄) · 타이틀 화면 · 설정 메뉴(사운드/저장 초기화/정보) · **전체 로컬 저장(백엔드 없이 새로고침 유지)** · itch.io 정적 빌드(`build:itch`, 상대 경로)
- **모바일 대응** — Capacitor(iOS/Android) 설정 + 반응형(1280×720 스테이지 통째 스케일·레터박스) + 터치 최적화 + 세로 모드 회전 안내
