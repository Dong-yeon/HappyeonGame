# 🏯 군웅록 (群雄錄) - 개발 문서

> 한국 삼국시대를 배경으로 한 방치형 액션 플랫포머 모바일 게임  
> 1인 개발 / 역사학과 출신 개발자의 고증 기반 프로젝트

---

## 📌 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 장르 | 방치형 액션 플랫포머 (Idle Action Platformer) |
| 배경 | 한국 삼국시대 (고구려 · 백제 · 신라) |
| 플랫폼 | 웹 먼저 출시 (itch.io) → 모바일 앱 (Capacitor 변환) |
| 개발 형태 | 1인 개발 |
| 레퍼런스 게임 | 메이플스토리 키우기 (액션 방치형 구조) |

---

## 🎮 핵심 게임 루프

```
[직업 선택]
    ↓ 화랑 / 싸울아비 / 개마무사 중 1개
[고정 맵 자동 전투]
    ↓ 하나의 맵 안에서 캐릭터가 왔다갔다하며 자동 전투
    ↓ 가장 가까운 적을 향해 이동 → 점프 → 공격
    ↓ 적은 맵 안에서 계속 스폰됨
[경험치 · 성장]
    ↓ 레벨업 → 스킬 강화
[장수 카드 수집]
    ↓ 역사 인물 카드로 캐릭터 능력치 강화
[다음 스테이지]
    ↓ 역사적 전투 배경 (살수대첩, 황산벌 등)
```

> ⚠️ 맵 구조 원칙: **무한 횡스크롤 아님**
> 하나의 고정된 맵 안에서 캐릭터가 자유롭게 이동하며 싸우는 방식.

---

## ⚔️ 직업 시스템 (Job/Class) — MVP 3직업

> 장수(역사 인물)는 캐릭터의 정체성이 아닌 **강화 레이어**로 활용
> 플레이어는 직업에 애정을 갖고, 장수 카드로 직업을 강화하는 구조

### 화랑 (花郞) — 신라
- **역사 근거**: 신라 화랑도. 황산벌의 관창처럼 적진을 단신 돌파하는 용맹한 전사
- **역할**: 고속 근접 딜러
- **전투 스타일**: 단검 + 대시 연속기
- **특기**: 적진 돌파, 회피기
- **플랫포머 동작**: 빠른 점프 + 공중 대시 공격
- **난이도**: ⭐ 입문

### 싸울아비 — 백제
- **역사 근거**: 백제 평민 전사를 뜻하는 순우리말. 중무장 보병
- **역할**: 탱커 + 근접 중딜러
- **전투 스타일**: 방패 + 환두대도
- **특기**: 피해 감소, 넉백 공격
- **플랫포머 동작**: 느리지만 묵직한 방패 찍기
- **난이도**: ⭐ 입문

### 개마무사 (鎧馬武士) — 고구려
- **역사 근거**: 사람과 말 모두 70kg 철갑으로 무장한 중장기병. 500명이 수나라 5만을 퇴각시킨 고구려 최정예
- **역할**: 느리지만 최강 한방 딜러
- **전투 스타일**: 기병창(삭) + 찰갑 중무장 (하마 상태)
- **특기**: 돌격 스킬, 착지 충격파
- **플랫포머 동작**: 육중한 점프 → 착지 시 광역 충격
- **난이도**: ⭐⭐ 보통

### 직업 비교

| | 화랑 | 싸울아비 | 개마무사 |
|--|------|---------|---------|
| 세력 | 신라 | 백제 | 고구려 |
| 속도 | 빠름 | 보통 | 느림 |
| 공격력 | 중 | 중 | 강 |
| 방어력 | 약 | 강 | 강 |
| 조작감 | 민첩·화려 | 묵직·안정 | 육중·파괴적 |

## 🃏 장수 카드 시스템 (강화 레이어)

- 장수는 플레이어블 캐릭터가 아닌 **수집·강화 아이템**
- 역사 인물 카드를 장착하면 직업별 스킬·능력치 강화
- 예시: `을지문덕의 전술서` → 개마무사 돌격 스킬 쿨타임 감소
- 예시: `계백의 최후` → 싸울아비 체력 임계치 이하 공격력 2배
- 예시: `김유신의 삼화령` → 화랑 다단 점프 추가

## 🃏 MVP 범위 (초기 출시 목표)

- [ ] 직업 3개 구현 (화랑 · 싸울아비 · 개마무사)
- [ ] 스테이지 5개 (역사적 배경)
- [ ] 자동 전투 + 점프·스킬 플랫포머 구현
- [ ] 장수 카드 10장 (직업별 3~4장)
- [ ] 광고 수익 모델 연동
- [ ] 웹 배포 (itch.io)

> 목표 개발 기간: **3~4개월**

## 🛠️ 기술 스택

### Frontend
- **React** — UI 전체 (패널, 메뉴, HUD, 장수 카드)
- **Phaser.js (Arcade Physics)** — 캐릭터 이동·점프·충돌·스킬 이펙트

### Backend
- **Spring Boot (Java)** — 게임 데이터 API 서버
- **SQLite** → 추후 MySQL 이전

### 배포
- **itch.io** — 웹 버전 초기 출시
- **Capacitor.js** — 모바일 앱 변환 (iOS / Android)

### 나중에 추가
- **Redis** — 세션 캐시 (멀티 대응 시)
- **Python FastAPI** — NPC AI, 밸런싱 분석

## 🗺️ 개발 로드맵

### Phase 1 — 프로토타입 (1~2개월)
- Phaser.js로 캐릭터 1개 (화랑) 이동·점프·공격 구현
- **고정 맵 1개** — 무한 스크롤 아님, 화면 안에서 캐릭터가 왔다갔다하며 전투
- 적은 맵 안에서 계속 스폰, 캐릭터는 가장 가까운 적을 자동 타겟팅
- React UI 기본 HUD 연결
- 자동 전투 루프 완성

### Phase 2 — MVP (2~3개월)
- 나머지 직업 2개 추가 (싸울아비, 개마무사)
- 스테이지 5개 완성
- 장수 카드 시스템 구현
- Spring Boot 백엔드 연결
- itch.io 웹 배포

### Phase 3 — 콘텐츠 확장 (이후)
- 장수 카드 30장으로 확대
- 역사 이벤트 시나리오
- 모바일 앱 출시 (Capacitor)
- 직업 추가 검토 (궁수, 무당 등)

### Phase 4 — SLG 전환 (장기)
- 멀티플레이어 도입
- 한반도 지도 영토 시스템
- 시즌제 도입
